turtle库

turtle（海龟）是turtle绘图体系的Python实现，是Python的标准库之一，使用RGB色彩模式。

空间坐标系

画布上以中心为原点的空间坐标系：绝对坐标&海龟坐标。

角度坐标系

画布上以空间x轴为0度的角度坐标系：绝对角度&海龟角度。

库引用

扩充Python程序功能的方式。

  1. 使用import保留字完成
import <库名>
<库名>.<函数名>（<函数参数>）
  1. 使用from和import保留字完成
from <库名> import <函数名>
from <库名> import *
<函数名>（<函数参数>）
  1. 使用as和import保留字完成
import <库名> as <库别名>
<库别名>.<函数名>（<函数参数>）
